# GIN-pytorch-DGL
An implementation of [GIN](https://openreview.net/forum?id=ryGs6iA5Km) model using [DGL](https://www.dgl.ai/) based on [Pytorch](https://pytorch.org/).

## Environment

- Python 3.7.3
- Numpy 1.151
- Pytorch 1.6.0+cu101
- DGL 0.5.1

## Dataset
NCI
